# Hostile Reviewer Response

The strongest criticism is fair: impedance-token semantics do not remove the need for a good token library. If the vocabulary lacks the right admittance or positive-work budget, the closed-loop token can be too sluggish or too conservative.

The v2 stress quantifies that boundary. The baseline token has force error 1.629 N and contact loss 0.019. Reducing admittance to 0.25x raises force error to 4.223 N and contact loss to 0.276. Tightening the positive-work budget to 0.05 J raises force error to 3.968 N and contact loss to 0.140.

The revised claim is therefore semantic and conditional: contact-rich skill tokens should denote physical interaction laws, but the learned or chosen token set must cover the required physical port laws.
