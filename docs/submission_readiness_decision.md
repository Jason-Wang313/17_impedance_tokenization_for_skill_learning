# Submission Readiness Decision

Decision: workshop-only / revise before main-conference submission.

The paper has a sharp mechanism: contact-rich action tokens should denote closed-loop impedance/admittance laws rather than kinematic chunks. The formal toy check and simulator support that semantic claim.

The paper is not ready as a main-conference robotics systems result. It uses a low-dimensional simulator, hand-specified token parameters, automated literature extraction, and no hardware contact validation.

Minimum next evidence for a stronger submission:

- Learn the token dictionary from demonstrations or interaction data.
- Compare against a stronger continuous impedance/admittance controller.
- Validate on a real or high-fidelity contact-rich manipulation task.
- Stress token selection, contact-frame estimation, force-sensor delay, and passivity margins.
